# Bilder för DadBod Elimination Protocol

Lägg alla bilder i denna mapp. Här är vilka bilder som behövs:

## Hero-sektion
- `hero-workout.jpg` eller `hero-workout.png` - Bild på pappa som tränar hemma

## Info Slides (Social Proof)
- `info-1-success.jpg` - Bild för första info-sliden (100,000 dads)
- `info-2-michael.jpg` - Bild för Michael's success story
- `info-3-lifestyle.jpg` - Bild för tredje info-sliden (real life)

## Bildfrågor - "How do you look right now?"
- `body-low-fat.jpg` - Low body fat
- `body-medium-fat.jpg` - Medium body fat
- `body-higher-fat.jpg` - Higher body fat
- `body-very-high-fat.jpg` - Very high body fat

## Bildfrågor - "What is your dream body?"
- `dream-lean-defined.jpg` - Lean and defined
- `dream-athletic-muscular.jpg` - Athletic and muscular
- `dream-bigger-strong.jpg` - Bigger and strong
- `dream-slimmer-healthy.jpg` - Just slimmer and healthier

## Bildformat
- Använd JPG eller PNG
- Rekommenderad storlek för hero: 800x1000px
- Rekommenderad storlek för info slides: 600x400px
- Rekommenderad storlek för body type images: 400x300px
- Optimera bilderna för web (använd t.ex. TinyPNG eller ImageOptim)
